## Development

Ensure global dependecies are installed:

- NodeJS

### Install local dependecies:

```sh
pnpm install
```

### Run the dev servers

```sh
pnpm dev
```

## Deployment

Netlify will deploy the site when:

- A commit is pushed to `master`
- A data change is made in Sanity
